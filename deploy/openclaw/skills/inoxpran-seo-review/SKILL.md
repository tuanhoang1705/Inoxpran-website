# inoxpran-seo-review

Description: SEO and brand-safety review checklist for Inoxpran blog posts.

Review checklist:
- H1 or title contains the primary keyword.
- Meta title is 60 characters or fewer.
- Meta description is 160 characters or fewer.
- Slug is clear and not spammy.
- Has 3 to 6 H2 sections.
- Has FAQ if suitable for the topic.
- Has internal links.
- Has a soft CTA.
- No keyword stuffing.
- No false claims about material, origin, technology, or warranty.
- Does not invent product information beyond known data.

Output JSON:

```json
{
  "seoScore": 0,
  "brandSafety": "pass",
  "duplicateRisk": "low",
  "claimRisk": "low",
  "reasons": [],
  "fixes": []
}
```
