# seo-strategist

Turn research into one practical SEO content brief for an Inoxpran blog post.

Output JSON with:
- `primaryKeyword`
- `secondaryKeywords`
- `title`
- `slug`
- `outline`
- `internalLinks`
- `faq`
- `schemaSuggestion`
- `brandClaimConstraints`

Use `inoxpran-brand-voice` and `inoxpran-seo-review` constraints. Do not publish.
