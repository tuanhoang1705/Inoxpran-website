# Inoxpran Combo Landing Page

Landing page tĩnh cho chiến dịch combo Inoxpran hè 2026. Mở trực tiếp `index.html` bằng browser là chạy được, không cần build tool.

## Cách dùng

- Mở `index.html` trong trình duyệt.
- Tailwind CSS, Google Fonts và Lucide icons đang dùng CDN, nên cần internet để render đúng style/icon.
- CTA Zalo hiện trỏ tới `https://zalo.me/0867024186`; thay bằng Zalo OA chính thức nếu có URL riêng.
